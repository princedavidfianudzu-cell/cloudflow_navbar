# ☁️ CloudFlow — Navbar Project

My first *HTML & CSS project*  
A clean and modern navigation bar built from scratch using HTML5 and CSS3.

## 🔍 Preview

CloudFlow is a modern navigation bar design for a cloud storage website.  
It includes:

- Brand logo
- Navigation links
- Login option
- Call-to-action button

The goal was to create a simple, clean, and user-friendly interface.

## 🛠️ Built With

- HTML5
- CSS3

## 📁 Project Files

| File | Description |
|------|-------------|
| `index.html` | Contains the structure of the navigation bar |
| `Try 1.css` | Contains all styling and design elements |

## ✨ Features

✅ Clean pill-shaped navigation header  
✅ Modern layout using CSS Flexbox  
✅ Smooth hover effects on links  
✅ Styled call-to-action button  
✅ Simple and readable code  
✅ Modern UI design  

## 🚀 How to Run

1. Clone or download this repository

2. Open the project folder

3. Open:

```
index.html
```

in your browser.

That's it! 🎉
